## ğŸ‰ How to Use mcp-tidy
After installation, using mcp-tidy is simple. Open your terminal and use the following commands:

1. **List Servers**
   ```
   mcp-tidy list
   ```
   This command displays all your current servers.

2. **Analyze Server Usage**
   ```
   mcp-tidy stats <server-name>
   ```
   Replace `<server-name>` with the name of the server you want to analyze. This will give you detailed statistics.

3. **Clean Up Unused Servers**
   ```
   mcp-tidy cleanup
   ```
   This command will prompt you to confirm the removal of unused servers.
